### 주의사항
대부분 운영체제 과제는 ARM 아키텍쳐에 친절하지않습니다.  
그렇기 때문에 pintos를 컴파일하기 위해서는 x86 아키텍쳐  컹퓨터가 필요합니다.  
즉 다음컴퓨터에서는 과제를 수행할수없습니다.
* Apple sillicon M1, M2, M3
* Apple sillicon M1, M2, M3에 설치한 Ubuntu
* Apple sillicon M1, M2, M3에 설치한 Windows
